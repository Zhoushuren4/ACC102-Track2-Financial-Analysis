## 1. Project Overview
This project is a Python-based data product that analyses key financial ratios of 10 major S&P 500 companies. The goal is to help accounting students and beginner investors understand corporate financial performance, including liquidity, leverage, profitability, and return levels through data analysis and visualisation.

## 2. Analytical Problem & Target Users
- **Analytical Problem**: Compare and evaluate the financial health of large listed companies using standard financial ratios.
- **Target Users**: Accounting students, finance learners, investors, and anyone interested in corporate financial performance.

## 3. Dataset Information
- **Content**: Financial indicators including Current Ratio, Debt-to-Equity Ratio, ROE, and Profit Margin.
- **Source**: Simulated financial dataset for educational use.
- **Access Date**: April 2026

## 4. Tools & Python Libraries
- Python 3
- Pandas
- Matplotlib
- Numpy

## 5. Key Analysis & Visualisations
1. ROE comparison bar chart
2. Profit Margin vs Current Ratio scatter plot
3. Descriptive statistics and financial performance ranking

## 6. How to Run
1. Clone or download this repository
2. Install required packages:
   pip install pandas numpy matplotlib
3. Run the script:
   python financial_analysis.py

## 7. Main Findings
- Microsoft and Visa show the highest profitability and ROE.
- Companies with higher current ratios have stronger short-term solvency.
- Retail and e-commerce companies have relatively lower profit margins.

## 8. Limitations & Improvements
- Limitations: Small sample size, cross-sectional data only.
- Improvements: Expand dataset, add time-series data, build interactive dashboard.
